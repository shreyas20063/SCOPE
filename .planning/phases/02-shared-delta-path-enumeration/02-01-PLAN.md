```python
def build_topology(sim, blocks, connections):
    """Programmatically build a block diagram topology.

    blocks: list of dicts, each passed as params to add_block action.
    connections: list of (from_block, from_port, to_block, to_port) tuples.
    """
    for b in blocks:
        sim.handle_action({"action": "add_block", "params": b})
    for fb, fp, tb, tp in connections:
        sim.handle_action({"action": "connect", "params": {
            "from_block": fb, "from_port": fp, "to_block": tb, "to_port": tp
        }})
```

   - junction2 -> gain(H2=0.3) -> adder2(port 1, sign "-")
   - NOTE: This creates two independent SISO loops. The graph determinant Delta should incorporate ALL loops from the full graph, so all 4 transfer matrix entries (G11, G12, G21, G22) must share the same denominator.

3. `test_1x2_shared_denominator(bdb_simulator)` -- Build a 1-input, 2-output system:
   - in1 -> junction1 -> gain(G1=3.0) -> out1
   - junction1 -> gain(G2=5.0) -> out2
   - No loops, Delta = [1.0] for both.
   - Assert both denominators are `[1.0]`.

4. `test_siso_still_works_after_refactor(bdb_simulator)` -- Regression test: simple SISO negative feedback.
   - in1 -> adder(port 0) -> gain(G=2.0) -> junction -> out1
   - junction -> gain(H=0.5) -> adder(port 1, sign "-")
   - The SISO result: G/(1+GH) = 2/(1+1) numerically. For operator-domain with gains only: num=[2.0], den=[2.0] (after normalization: num=[1.0], den=[1.0]). Actually with low-power-first: forward path gain P1 = 2.0 -> (num=[2.0], den=[1.0]). Loop gain L1 = G*H*(-1) = -1.0. Delta = 1 - (-1) = [2.0]. Cofactor Delta1 = [1.0]. TF = 2*1 / 2 = 1.0. After normalization: num=[1.0], den=[1.0].
   - Wait -- signs: the adder handles the negative sign, so the loop gain traverses adder->G->junction->H->adder. The sign at the adder's port 1 is "-", so loop gain = G*H*(-1) = 2*0.5*(-1) = -1. Delta = 1 - (-1) = 2. So TF = P1*Delta1/Delta = 2*1/2 = 1.0.
   - Actually more carefully: the loop gain as computed by `compute_loop_gain` includes the sign from the adder port. So the raw loop gain (before the Mason formula sign) is -1.0. Then Delta = 1 - sum(L_i) = 1 - (-1) = 2. The forward path gain P1 = 2 (through adder -> G -> junction, adder contributes sign "+" on port 0). So TF = P1/Delta = 2/2 = 1.
   - Assert `_tf_result["numerator"]` and `_tf_result["denominator"]` are both `[1.0]` (within tolerance). Also assert `_tf_result["num_forward_paths"] == 1` and `_tf_result["num_loops"] == 1`.
